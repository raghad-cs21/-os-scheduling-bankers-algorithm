# OS Scheduling & Banker's Algorithm Simulation

This project is a C++ simulation of common CPU scheduling algorithms and the Banker's Algorithm used for deadlock avoidance in operating systems.

## Features
- Simulation of CPU Scheduling algorithms
- Implementation of Banker's Algorithm
- Demonstrates operating system process management concepts

## Technologies Used
- C++
- Operating Systems concepts
- Algorithm simulation

## Purpose
This project was developed as part of an Operating Systems course to demonstrate scheduling strategies and safe resource allocation techniques.
